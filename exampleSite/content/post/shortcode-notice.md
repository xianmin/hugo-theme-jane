## normal use
`Note` example:

```shortcode
{{%/* notice note */%}}
A notice disclaimer
{{%/* /notice */%}}
```

**Result:**
{{% notice note %}}
A notice disclaimer
{{% /notice %}}


You could **custom title** :


```shortcode
{{%/* notice note 笔记 */%}}
A notice disclaimer
{{%/* /notice */%}}
```

{{% notice note 笔记 %}}
A notice disclaimer
{{% /notice %}}
